# Connect Four

A terminal-based Connect Four game with two-player and single-player (vs AI) modes.

## Features

- 6x7 board with standard Connect Four rules
- Two-player local mode
- AI opponent with basic positional evaluation
- Win detection for horizontal, vertical, and diagonal connections
- Draw detection

## How to Run

```
python connect_four.py
```

## Gameplay

Players take turns dropping pieces into columns. First to connect four in a row (horizontally, vertically, or diagonally) wins.
